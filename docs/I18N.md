# i18n — Çeviri altyapısı

Varsayılan dil: **tr**. Dil değiştirici UI yok (ileride preferences + switcher).

## Frontend (`@alatax/shared`)

- Config: `packages/shared/src/i18n/index.ts`
- Çeviriler: `packages/shared/src/i18n/locales/tr/{common,auth,validation}.json`
- Her SPA `main.tsx` içinde: `import '@shared/i18n'`
- Hook: `import { useTranslation } from '@shared/i18n'`

### Yeni çeviri ekleme (örnek)

1. `packages/shared/src/i18n/locales/tr/common.json` içine anahtar ekle:

```json
{
  "save": "Kaydet",
  "employees": {
    "title": "Personel Listesi"
  }
}
```

2. Bileşende kullan:

```tsx
import { useTranslation } from '@shared/i18n';

export function EmployeesHeader() {
  const { t } = useTranslation('common');
  return <h1>{t('employees.title')}</h1>;
}
```

Namespace'li kullanım: `t('auth:forgot.title')` veya `useTranslation(['auth','validation'])`.

**Kural:** Yeni UI metni hardcode edilmez; `t()` zorunlu (`.cursorrules`).

## Backend (Laravel)

- Locale: `config/app.php` → `locale` / `fallback_locale` = `tr`
- Dosyalar: `lang/tr/{validation,auth,passwords,messages,pagination}.php`
- Kullanım: `__('messages.mail.reset_password_subject', ['app' => config('app.name')])`
- Validation mesajları otomatik `lang/tr/validation.php` üzerinden gelir.
